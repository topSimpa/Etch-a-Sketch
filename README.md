# Etch-a-Sketch(Web-version)
A web mimick of a classic "Etch-A-Sketch" drawing toy.
Featuring dynamic grid resizing and hover-based drawing built
with vanilla Javascript to practice DOM manipulation and UI
layout logic. A project in following through with the The Odin Project foundation
see: [Etch-A-Sketch](https://www.theodinproject.com/lessons/foundations-etch-a-sketch)

## Overview
This project builds a grid systems that allows for user to draw shapes by interacting with it.<br>
it uses Javascript driven UI, DOM manipulation and flexbox layout to produce an interactive web app.

see: [live Demo](https://topsimpa.github.io/Etch-a-Sketch/)

## Learning Objectives
- Pracitce DOM Manipulation using Javascript
- Work with event handling for user interactions
- Reinforce problem-solving and debugging skills
- Apply flexbox layout techniques

## Technologies Used
- HTML5
- CSS3 (Flexbox layout Algorithm)
- Javascript (Vanilla)
